# Shri Digambar Jain Mandir - Hastinapur (Jambudweep)

A premium, full-stack web application for the **Digambar Jain Trilok Shodh Sansthan**, Hastinapur. This website serves as a digital archive and informational hub for the iconic Jambudweep Tirth, established under the guidance of **Pujya Ganinipramukh Aryika Shri Gyanmati Mataji**.

## 🌟 Features

- **Immersive Design**: High-fidelity UI built with React, Tailwind CSS, and Framer Motion.
- **Spiritual Heritage**: Detailed sections for the 108ft Statue of Ahimsa, Sumeru Parvat, and the Lotus Temple.
- **Full-Stack Integration**: 
  - **Frontend**: Vite-powered React application with smooth transitions and responsive layouts.
  - **Backend**: Laravel API handling contact form submissions and spiritual inquiries.
  - **Database**: MySQL integration for persistent data storage.
- **Bilingual Support**: Comprehensive information in both English and Hindi.
- **Interactive Gallery**: High-resolution visual tour of the holy site.

## 🛠️ Tech Stack

- **Frontend**: React 19, Tailwind CSS v3, Framer Motion, Lucide React.
- **Backend**: Laravel 11 / Node.js Express.
- **Database**: MySQL.
- **Styling**: Custom Design System with "Deep Crimson" and "Warm Gold" palette.

## 🚀 Getting Started

### Prerequisites

- Node.js (v18 or higher)
- PHP 8.2+ & Composer (for Laravel backend)
- MySQL Server

### Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/Aradhy20/jain-mandir-website.git
   cd jain-mandir-website
   ```

2. **Frontend Setup**:
   ```bash
   npm install
   npm run dev
   ```

3. **Backend Setup**:
   ```bash
   cd backend
   composer install
   cp .env.example .env
   php artisan key:generate
   php artisan migrate
   php artisan serve
   ```
